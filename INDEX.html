✅ Deployment Checklist

Before You Start

  • ✓ AWS account created
  • ✓ AWS CLI installed
  • ✓ Node.js 18+ installed
  • ✓ Amplify CLI installed
  • ✓ Git installed

During Deployment

  • ✓ Run deploy.ps1 (or deploy.sh)
